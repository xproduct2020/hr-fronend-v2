# hr-fronend-v2

Next.js frontend for job seeker tracker.

## Setup

- `npm install`
- copy `.env.example` to `.env.local`
- `npm run dev`

## Environment

- `NEXT_PUBLIC_API_URL`
- `NEXT_PUBLIC_GOOGLE_CLIENT_ID`

## Pages

- `/` Job seeker auth page
- `/employer` Employer auth page
- `/admin` Admin auth page
- `/dashboard` Shared dashboard after login

## Vercel

Deploy directly as Next.js project.
